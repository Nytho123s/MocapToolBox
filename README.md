# MocapToolBox
A Tool that helps you clean mocap data directly in Maya Autodesk. (WIP)  
Maya version (2025.2)  
  
There is a Drag And Drop Install located here : C:\TOOLS_Production\mocaptoolbox\cg\Maya\addons\mocaptoolbox\Drag_and_Drop_to_install.MEL

Make sure the mocaptoolbox folder is located in C:\TOOLS_Production\mocaptoolbox, if not, the tool won't work

The IKchain system is not complete yet and the colors won't change.
